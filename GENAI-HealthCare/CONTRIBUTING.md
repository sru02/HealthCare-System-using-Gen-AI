# Contributing to GenAI Healthcare Assistant

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/genai-healthcare-assistant.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Set up the project locally following the [README](README.md)

## Making Changes

- Follow the existing code style
- Keep commits small and focused
- Write clear commit messages: `feat: add appointment reminder`, `fix: symptom API timeout`

## Submitting a Pull Request

1. Push your branch: `git push origin feature/your-feature-name`
2. Open a Pull Request against `main`
3. Fill in the PR template describing what you changed and why
4. Wait for review

## Reporting Bugs

Use the [Bug Report](.github/ISSUE_TEMPLATE/bug_report.md) issue template.

## Suggesting Features

Open an issue with the title `[Feature Request]: your idea` and describe the use case.
